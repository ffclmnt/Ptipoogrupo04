# SOBRE

Trata-se de um projeto de gestão de dados de uma universidade. Construímos um sistema de cadastro de alunos, professores e fornecedores de uma universidade. O projeto foi elaborado no 3º semestre do curso de Análise e Desenvolvimento de Sistemas do SENAC, como parte da disciplina de Projeto Integrador.

# ESTRUTURA

1. **Protótipos da Interface**: inclui o layout e UI;
2. **diagrams_phase1**: inclui os diagramas de caso de uso e de classes, feitos na primeira etapa do projeto;
3. **sql**: inclui o banco de dados criado para esse projeto, com base na documentação da primeira fase do projeto;
4. **docs**: inclui as descrições dos fluxos do usuário ao longo de uma jornada completa.**
